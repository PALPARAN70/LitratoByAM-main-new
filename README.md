Welcome to LitratoByAM Website!

LitratoByAM is a photobooth business run by Mikee & Ace


To run our program follow these steps:

-Make a folder
-Open folder via VSCode
-Go to terminal and run "git clone (repo link)"
-Open terminal and run cd LitratoByAM-main-new then enter
-Run cd backend, then run npm i, then run npm run dev
-Open new terminal and cd LitratoByAM-main-new, run npm i then npm run dev
